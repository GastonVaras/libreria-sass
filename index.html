El estilo de esta landing page fue hecho
unicamente añadiendo clases a los elementos!

Las clases a utilizar se generan con SASS.

Boton inactivo

Diseño reactivo

Por medio de clases establecemos un sistema de grilla para las diferentes resoluciones, prueba ajustando la página!

Damos estilo al div "container" con las clases "bg-secondary-light-9", establece su color a la $variable creada mezclada con un 90% de blanco. Podria ser "bg-primary-dark-1". Esta amplia gama de opciones la tenemos gracias a SASS usando un mapa de color.

Otros elementos como margen, padding, colores, botones y mucho mas fueron creados utilizando mixins, funciones y elementos integrados de SASS para tener una libreria con mas de 4000 líneas de código!

Sin embargo para ahorrar espacio por medio de gulp podemos filtrar esta enorme libreria a las clases que fueron efectivamente utilizadas, de este modo hacemos el codigo final facil de interpretar.
